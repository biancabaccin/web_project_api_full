# Tripleten Around the U.S.

Around the U.S. é um projeto do programa de Web Development da TripleTen, focado na criação de uma aplicação React interativa onde os usuários podem adicionar, remover e curtir fotos; além de editar dados do perfil, como nome, descrição e avatar.

Nesta etapa do projeto, foi implementado um sistema completo de registro e autenticação de usuários, com rotas protegidas, integração com API externa e armazenamento de token.

# Principais Recursos Utilizados:

- HTML
- CSS (Flexbox, Grid Layout, Responsividade)
- JavaScript (Manipulação do DOM, Validação de Formulários, Objetos e Classes)
- React, React Router e Vite
- Organização Modular
- API REST
- Autenticação e Autorização (JWT)
- ProtectedRoute
- LocalStorage
